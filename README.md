### Installation:
 - Download Zip
 - Import to [Replit](https://replit.com) 
 - Customize The Files You Want (in folder presences) 
 - Create "TOKEN" In Secret Variable & Put Your Discord Token In The Values 
 - Run The Project
<br/>

### Features:
 - Listening Status (Spotify)
 - Playing Status
 - Streaming Status
